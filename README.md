# Basic CRUD app demo

## Objectives

Demo the basic CRUD actions in a Rails app

- view all of the songs
- show 1 song
- create a new song
- edit/update a song
- delete a song

## Instructions

* Clone this repo,
* run `bundle`
* run `rake db:create db:migrate db:seed`
* run `rails server`
* Open a browser to `http://localhost:3000/`
